# IPL Match Analytics & Performance Insights (2008–2023)

Comprehensive IPL data analytics project focused on uncovering team performance trends, player statistics, venue behavior, and match-winning factors using Python, SQL, Power BI, and Excel.

## Project Overview

This project performs end-to-end exploratory data analysis (EDA) on Indian Premier League (IPL) datasets from 2008 to 2023. The objective is to analyze historical match data and generate actionable insights related to team consistency, player performance, venue impact, toss decisions, and seasonal trends.

The project combines data preprocessing, statistical analysis, visualization, and dashboard reporting techniques commonly used in real-world sports analytics workflows.

## Objectives

- Analyze historical IPL match performance across seasons
- Evaluate team and player consistency metrics
- Identify venue-specific winning patterns
- Study toss decisions and their impact on match outcomes
- Generate interactive visual insights and performance reports
- Apply analytical techniques for sports data interpretation
